# Banking Executive KPI Simulator

Python generator for synthetic banking executive KPIs.

This project generates daily banking performance metrics that can be used to build executive dashboards in Tableau, Power BI, or other BI platforms.

## Generated Metrics

- Revenue
- Profit
- Loan Book
- Deposit Book
- Net Interest Margin (NIM)
- Cost-to-Income Ratio
- Non-Performing Loan Ratio (NPL)
- Customer Growth
- Customer Churn
- Digital Logins
- Branch Visits

## Dashboard Use Cases

Executive banking dashboards focused on:

- Financial Performance
- Lending Growth
- Deposit Growth
- Operational Efficiency
- Credit Risk
- Customer Growth
- Digital Adoption

## Technology

- Python
- Pandas
- NumPy

All data is synthetically generated and intended for educational and portfolio purposes only.
